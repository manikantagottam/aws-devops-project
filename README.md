# DevOps Full Stack Project 🚀

This is a simple full stack application using Backend and Frontend, containerized with Docker and CI/CD

## 📌 Project Overview
This project contains:
- Backend (Node.js + Express)
- Frontend (HTML)
- Docker setup

## 🛠️ Technologies Used
- Node.js
- Express
- Docker
- GitHub

## 📂 Project Structure

backend/
  - Dockerfile
  - package.json
  - index.js

frontend/
  - Dockerfile
  - index.html

## ▶️ How to Run

1. Clone the repository
2. Run:
   docker-compose up --build
3. Open browser:
   http://localhost:3000
